# Frankenstein V1


## Descrição
O **InitFrame** nasceu com o nobre propósito de criar uma estrutura leve e fácil, que poderia ser usada por desenvolvedores iniciantes.
Entretanto, ele possui limitações fortes (um sistema de roteamento fraco, renderização de views precária). Na realidade, ele acabou por ser usado somente em um trabalho de faculdade (e tive que realizar modificações estruturais para atender as necessidades do projeto).

Esses fatores culminaram no projeto **Frankenstein**, que, basicamente, surgiu com o propósito de melhorar o **InitFrame**. Entretanto, o projeto me levou para um caminho diferente. Apesar de ter nascido como uma evolução **InitFrame**, o **Frankenstein** passou a ser um projeto separado, herdando muito pouco de seu antecessor.

Ainda há muitas coisas a serem feitas e talvez tudo mude e um outro projeto nasça, mas atualmente meus esforços estão aqui.
